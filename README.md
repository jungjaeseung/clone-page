## Initial Step

First, install node modules:
```bash
yarn install
```

## Dev Mode

run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build Proudct

build server:

```bash
yarn build
```

Then the source files will be created: "./.next"

## Run the product

start server:

```bash
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
